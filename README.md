# dish shell

A shell completely written in rust

## Things to consider
- x or exit to exit, c or clear to clear

## Build instructions
First, clone this repository with git:

```bash
git clone --recursive https://codeberg.org/D1SH/Dish dish
cd dish
```
Next, build the source code with one of these commands

```bash
make build-x86_64
or
make build-i486
```
